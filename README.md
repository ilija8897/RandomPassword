# RandomPassword

В целом для новичка хорошая вкатывающая задачка. Плавно подводит к самостоятельной реализации функции проверки качеста пароля.
- Возможно будет полезно добавить необходимость коммита после каждого этапа.
- Если студент создаст репозиторий без файла Readme например у него не будет доступна кнопка Codespace и надо будет загрузить файл вручную для начала. Возможно этот момент тоже стоит подсветить. У некоторых ребят такие моменты вызывали ступор иногда, особенно без знания англ.
